# SPOTI-APP
```
Just for test the Spotify API
```

# Requeriments
```
Spotify needs an access token to use the API.
Please, read the docs here: https://developer.spotify.com/documentation/web-api/
```

# Caracteristicas
```
Muestra de los últimos 20 releases en Spotify en tarjetas
Información sobre el artista, datos, y los 10 tracks más relevantes. También con un preview de las canciones
Buscador integrado
```
# Important
```
Para utilizar la API de Spotify se requiere de un token personalizado:
* Por ahora, se genera con un botón por interfaz, elegí esta forma ya que fue un proyecto sólo de testeo y rápido
```
